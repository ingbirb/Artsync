 Example Filled-In (Artist-Friendly)
Application #1
Application Name: ArtSync

Problem Trying to Solve:
Artists struggle to organize multiple creative projects and track progress across different mediums.
Purpose / Description:
A project management app designed for artists to log, track, and share progress on artworks.
Target Users:
- Student Artists: Need structure for assignments and deadlines.
- Freelance Illustrators: Manage multiple client projects at once.
- Hobbyists: Want motivation to finish personal art pieces.
Main Functions / Features:
- Register – Create account with art focus.
- Login – Secure access to project tracker.
- Dashboard – Shows active projects, deadlines, and progress bars.
- Profile – Stores portfolio, style preferences, and completed works.
- Logout – End session securely.
- Inspiration Board (upload sketches, references).
- Progress Tracker (visual bar for each artwork).//optional
System Flow (Wireframe Logic):
- Launch Screen → Login / Register Page.
- Registration Flow: Email → Choose Art Category → Profile Setup → Dashboard.
- Dashboard: Active Projects, Deadlines, Inspiration Board.
- Project Flow: Dashboard → Add Project → Track Progress → Save Updates.
- Profile Flow: View/Edit Portfolio, Completed Works.
- Logout Flow: Secure exit.
